ポッドキャスト「くもキャスト」の @gaaamii と @karur4n がISUCONに参加するための準備や訓練を行うリポジトリです

## ISUCONってなんだ
ISUCON（Iikanjini Speed Up Contest）はLINE主催のウェブアプリチューニングのコンテストです。優勝すると100万円,2位だと30万円もらえるらしいです。
